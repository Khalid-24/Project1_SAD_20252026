## 📌 Project Overview
This project is developed as part of the **System Analysis and Design (SECD2613)** course.  
It focuses on designing a **Student Health Management System (SHMS)** for Universiti Teknologi Malaysia (UTM), specifically for **Pusat Kesihatan Universiti (PKU)**.

The system aims to improve healthcare services by providing a digital platform where students can:
- Book appointments online  
- Access their medical records  
- Communicate with healthcare staff  

At the same time, it helps clinic staff manage patient data efficiently, reduce manual work, and improve overall service quality.

---

## 🎯 Project Objectives
- Improve appointment scheduling and reduce waiting time  
- Enhance accessibility of student health records  
- Improve communication between students and staff  
- Provide 24-hour access to emergency support information  
- Increase efficiency in healthcare service management
